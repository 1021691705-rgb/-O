### 发现的问题

- Obsidian 会因为点击空链接生成 0 字节笔记，需要定期清理。
- 模板里的空链接占位会污染图谱，已改成普通空列表项。
- `.obsidian/graph.json` 属于图谱状态文件，不适合作为知识内容提交。
- `core.autocrlf=true` 会带来换行提示，仓库已用 `.gitattributes` 固定文本文件换行策略。
- 已经被 Git 跟踪的本地状态文件，仅写入 `.gitignore` 不够，需要 `git rm --cached` 让忽略规则生效。

### 下次继续

- 整理 `OpenClaw` 后续主题：复刻手册、bindings 配置、多 Agent 隔离验证、长连接与 Webhook 对比。
- 同步 GitHub 前，先确认提交边界，避免混入 Obsidian 本地状态文件和 canvas 草稿。
- 如果要同步，优先只添加本轮 `.md`、`.gitignore`、`.gitattributes`，不要添加 `.obsidian/graph.json` 和未确认草稿。
