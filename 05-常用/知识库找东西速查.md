# 知识库找东西速查

## 一句话总结

需要资料时，先判断自己要找的是“入口、流程、工具、视觉词块、项目记录还是临时资料”，再从对应入口进入，不要只靠模糊搜索。

## 最快找法

### 1. 在 Obsidian 里找

- 按 `Ctrl + O`：按文件名快速打开，例如输入“反推模板”“资源总览”“Verysync”。
- 点左侧文件夹图标：按目录展开文件。
- 点左侧放大镜图标：全文搜索正文内容。
- 从 [[首页]] 进入：适合不知道从哪里开始时。

### 2. 让 Codex 帮你找

可以直接这样说：

```text
查知识库：我想找【主题】，帮我读最相关的 1-5 篇笔记。
```

或者：

```text
按知识库来：我现在要做【任务】，你先判断该读哪些笔记。
```

## 找不到时怎么说

```text
我找不到【主题】，你帮我在知识库里查一下，告诉我应该看哪几篇。
```

```text
这个东西应该入库吗？如果应该，帮我放到正确位置。
```

```text
我现在要做【任务】，你按知识库流程给我下一步。
```
