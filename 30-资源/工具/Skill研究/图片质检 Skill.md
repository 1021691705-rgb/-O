# 图片质检 Skill

## 一句话总结

检查设计方案或生成图片是否符合产品真实性、品牌一致性、平台规则和转化目标，并给出可执行返工建议。

## 参考来源

- Nielsen Norman Group UX / 设计评审方法：评审要基于明确标准，而不是个人喜好。
- Amazon 等电商平台图片规则：图片不能误导消费者，产品展示要准确。
- AI设计公司质检总监规则：涉及真实性和产品结构时拥有最终否决权。

## 触发场景

- 用户使用 `/质检`。
- 生成图出现结构变形、风格跑偏、卖点不清。
- 交付前需要检查方案是否可用。

## 输入

- 原始需求。
- 产品资料。
- 品牌标准。
- 方案或图片。
- 平台规则。
- 用户确认过的不可改动点。

## 输出

- 结论：通过 / 暂不通过 / 需要补资料。
- 问题清单。
- 严重程度。
- 修改建议。
- 返工任务。
- 下一轮 prompt 修改点。

## 质检维度

| 维度 | 检查问题 |
|---|---|
| 产品真实性 | 结构、比例、部件、材质是否被改错 |
| 品牌一致性 | 色彩、气质、视觉语言是否统一 |
| 平台适配 | 尺寸、文字、合规要求是否满足 |
| 转化逻辑 | 卖点是否清楚，用户是否知道为什么买 |
| 成本效率 | 是否会造成重复返工或素材浪费 |
