# 系统 Skill 总览

## 一句话总结

系统 Skill 负责知识库、Agent、同步、整理、检索和维护，是让整个知识库长期可用的底层工作流。

## 使用场景

- 入库、整理、同步、查库、修断链、维护目录。
- 给 Codex 设定知识库读取规则。
- 做跨项目知识库管理。
