# 图片裁切 Skill

## 一句话总结

把已有图片按目标比例和尺寸进行非破坏性裁切与导出，优先保证主体完整、比例准确，不允许暗中拉伸变形。

## 触发场景

- 用户说“裁图”“裁一下”“改成 9:16 / 16:9 / 1:1”。
- 生成图比例接近目标比例，但需要严格适配平台尺寸。
- 图片要作为手机壁纸、小红书封面、详情页模块、广告图或飞书预览图。

## 输入

- 源图片路径。
- 目标比例，例如 `9:16`、`16:9`、`1:1`、`3:4`。
- 目标尺寸，例如 `2160×3840`、`1080×1920`。
- 主体保护要求，例如“猫要完整”“产品不能被裁掉”“上方留字空间”。
- 是否覆盖原图：默认不覆盖。

## 输出

- 一张严格目标比例的裁切图。
- 如需要，再输出一张目标分辨率版本。
- 明确说明源尺寸、裁切尺寸、导出尺寸和是否拉伸。

## 标准流程

```text
读取源图尺寸
-> 判断源图比例和目标比例
-> 选择裁切方向
-> 保护主体和关键区域
-> 居中或按主体位置裁切
-> 导出严格目标比例版本
-> 如需 4K，等比放大到目标尺寸
-> 校验宽高和比例
-> 保留原图，不覆盖
```

## 裁切规则

- 只裁切，不强行拉伸。
- 默认居中裁切；如果主体偏边，优先保护主体。
- 目标比例必须精确，例如 `9:16 = 0.5625`。
- 4K 竖版默认导出 `2160×3840`。
- 4K 横版默认导出 `3840×2160`。
- 输出文件名要带用途和比例，例如 `cat-sleeping-in-sun-cropped-9x16-4k.png`。

## 质检清单

- [ ] 源图保留。
- [ ] 输出图比例准确。
- [ ] 主体没有被关键裁掉。
- [ ] 没有横向或纵向拉伸变形。
- [ ] 文件名能看出用途、比例和版本。
- [ ] 如用户要求 4K，输出尺寸符合 4K 目标。
