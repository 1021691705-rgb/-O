## 使用规则

- 只记录网站入口和用途，不记录登录信息。
- 涉及 API key 的网站，只把 key 保存在安全配置或密码管理工具里。
- 涉及客户素材、品牌素材和未公开图片时，优先使用本地工具。
- 第三方插件接入 Codex 前，先确认它会读取、同步、上传哪些数据。
- 购买 API 前，先小额测试稳定性和延迟，不直接用于正式生产链路。
