# 微力同步 Verysync

## 一句话总结

微力同步 Verysync 是一个用于多设备文件同步的工具，可作为个人资料、工作素材、NAS 和手机端之间实时同步的候选方案。

## 来源

- 官网下载页：https://www.verysync.com/download.html
- 入库日期：2026-07-08
- 用户补充：实时同步、国产、免费、支持手机端同步、支持 NAS。
- 访问情况：官网页面可访问；普通 PowerShell 请求出现 TLS 信任问题，使用浏览器或忽略证书校验后可读取下载页。

## 官网确认的信息

- 下载页标题为“免费下载”。
- 页面主文案为“同步所有数据、跨越您的全部设备”。
- 下载页按平台分为：桌面设备、NAS 系统、移动设备、路由器、Docker。
- 桌面和系统平台包括：Windows、macOS、Linux、FreeBSD。
- 移动端包括：iOS、Android。
- NAS 相关入口包括：Synology、QNAP。
- Docker 相关入口包括：命令行安装、群晖 Docker 安装。

## 适合解决什么问题

- 多台电脑之间同步项目资料、知识库、图片素材和交付文件。
- 手机、电脑、NAS 之间同步常用文件。
- 不想完全依赖单一网盘时，作为私有同步或备份链路候选。
- 有 NAS 或本地设备长期在线时，用来搭建自己的资料同步节点。

## 使用建议

1. 先用非敏感测试文件夹验证同步速度、冲突处理和断网恢复。
2. 如果用于 Obsidian 知识库，先确认 `.git`、`.obsidian/workspace.json`、缓存目录和大文件是否需要排除。
3. 如果和 GitHub 同步同时使用，要避免多端同时编辑同一个 Markdown 文件造成冲突。
4. 手机端同步前先测试中文文件名、图片附件和大文件表现。
5. NAS 部署前先确认对应型号、系统版本和 Docker 权限。

## 风险和注意事项

- 文件同步工具可能把误删、覆盖和冲突同步到所有设备，首次使用要保留独立备份。
- Obsidian 知识库同时使用 Git 和实时同步时，可能出现重复冲突文件或未完成写入状态。
- 客户素材、未公开设计稿和账号配置文件进入同步目录前，要确认访问权限和同步范围。
- 免费状态、平台支持和下载入口可能变化，正式部署前以官网当前页面为准。
