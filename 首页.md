# AI设计公司知识库

这里是 AI设计公司总控项目的知识入口。目标是把设计流程、岗位协同、Agent Skill、工具经验和复盘结论沉淀成可复用资产。

## 主要入口

| 区域 | 用途 | 入口 |
|---|---|---|
| 收集箱 | 临时资料、待整理链接、截图和灵感 | [[00-收集箱/收集箱]] |
| 项目 | 正在推进或已复盘的项目 | [[10-项目/项目总览]] |
| 领域 | 长期经营的业务领域 | [[20-领域/领域总览]] |

| 每日记录 | 每日记录和零散过程 | [[40-每日记录/每日记录]] |

## 常用动作

- 新资料先进 [[00-收集箱/收集箱]]。
- 可复用方法沉淀到 [[30-资源/资源总览]]。
- Agent / Skill 相关内容归到 [[30-资源/工具/Skill研究/Skill研究总览]]。
- 项目做完后补 [[30-资源/工具/Skill研究/项目复盘 Skill]]。
- 知识库变更默认同步 GitHub。

## 整理原则

- 一个主题只保留一个主入口。
- 总览页保持短，只负责导航。
- 具体经验写进具体笔记。
- 空链接要么补内容，要么删除。
